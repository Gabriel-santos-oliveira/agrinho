# Agro Forte, Futuro Sustentável 🌱

Projeto educativo focado no tema **Agrinho**, explorando a tecnologia no campo e a preservação ambiental.

## ✨ Funcionalidades
- **Saudação Dinâmica**: Interação direta com o usuário via JS.
- **Modo Escuro Nativo**: Troca de temas utilizando variáveis CSS.
- **Acessibilidade**: Controle deslizante para alterar o tamanho da fonte.
- **Responsividade**: Layout adaptável para tablets e celulares.

## 🛠️ Tecnologias
- HTML5 (Semântico)
- CSS3 (Flexbox/Grid/Variables)
- JavaScript (DOM Manipulation)

## 📁 Estrutura
- `index.html` (Principal)
- `/css/style.css` (Estilos)
- `/js/script.js` (Lógica)
