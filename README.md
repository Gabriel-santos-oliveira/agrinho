# Agro Forte, Futuro Sustentável 🌱

Este projeto foi desenvolvido para o tema **Agrinho 2026**, focado em demonstrar o equilíbrio necessário entre a alta produtividade do campo e a preservação ambiental.

## 🎯 Objetivo
Mostrar que a tecnologia (Agro 5.0) é a maior aliada da sustentabilidade, permitindo produzir mais com menos recursos e maior respeito à biodiversidade.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estrutura semântica (header, main, section, article, footer).
- **CSS3**: Layouts modernos com Flexbox e CSS Grid, variáveis para Modo Escuro e Media Queries para responsividade.
- **JavaScript**: Manipulação funcional do DOM, lógica de acessibilidade e personalização de interface.

## 🚀 Funcionalidades
- **Saudação Personalizada**: Interação via JS que captura o nome do usuário.
- **Modo Escuro**: Alternância de tema para conforto visual.
- **Controle de Fonte**: Botões para aumentar/diminuir o texto via script.
- **Navegação Fluida**: Links internos e externos para fontes oficiais (EMBRAPA, FAO).

## 📂 Como utilizar
1. Clone o repositório.
2. Certifique-se de manter a estrutura de pastas (`css/`, `js/`, `assets/`).
3. Abra o arquivo `index.html` em qualquer navegador moderno.
