# Projeto AgroForte: Futuro Sustentável 🌾

Este projeto foi desenvolvido para o tema **Agrinho**, focando no equilíbrio entre a alta produtividade agrícola e a preservação ambiental.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estrutura semântica.
- **CSS3**: Layout responsivo (Flexbox) e variáveis para Modo Escuro.
- **JavaScript**: Manipulação dinâmica do DOM e lógica de interação.
- **Canva/Paint**: Customização das imagens de fundo e ícones.

## 🎯 Objetivo
Demonstrar como a tecnologia (Agro 4.0) pode ser usada para monitorar recursos e reduzir impactos ambientais, promovendo uma consciência sustentável desde o ambiente escolar até o campo.

## 🛠️ Como usar
1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Insira seu nome para uma experiência personalizada.
4. Clique nos cards para ser direcionado a fontes oficiais sobre o Agro.
