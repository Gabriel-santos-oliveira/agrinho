# Agro Forte, Futuro Sustentável 🌱

## 🎯 Objetivo do Tema (Agrinho)
Este projeto foi desenvolvido com o objetivo de conscientizar sobre a importância do equilíbrio entre a alta produtividade agrícola (Agro Forte) e a preservação do meio ambiente (Futuro Sustentável). A página demonstra como a tecnologia e as práticas sustentáveis caminham juntas para garantir alimento para o mundo sem esgotar nossos recursos naturais.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização com Flexbox, Media Queries (Responsividade) e transições suaves.
* **JavaScript:** Interatividade com o usuário (Modo Escuro, saudação personalizada e manipulação do DOM).

## 🚀 Instruções de Uso
1. Faça o download ou clone este repositório.
2. Certifique-se de que a estrutura de pastas (`/css`, `/js`, `/img`) está mantida.
3. Adicione suas imagens autorais (criadas no Canva/Paint) na pasta `/img` e renomeie-as conforme os caminhos no arquivo `index.html`.
4. Abra o arquivo `index.html` em qualquer navegador web moderno (Chrome, Edge, Firefox).
5. Interaja com a página:
   - Digite seu nome no formulário para uma saudação.
   - Clique no botão "Modo Escuro" no topo da página.
   - Teste a página em diferentes tamanhos de tela (celular e PC).
   - Clique nos botões de links externos.
